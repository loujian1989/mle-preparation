# Behavioral Interview Prep — Staff MLE

---

## STAR Format (Staff-Level Calibration)

| Component | What Senior says | What Staff says |
|---|---|---|
| **Situation** | "My team was working on X" | "The org was facing X — this affected 3 teams and a $YM product line" |
| **Task** | "I was asked to fix X" | "I identified that X was the wrong problem; the real issue was Y. I took ownership of defining scope." |
| **Action** | "I wrote the code / model" | "I built the case with data, aligned 3 PMs, unblocked 2 teams, and made the call others were avoiding" |
| **Result** | "It improved metric by X%" | "X% metric lift + Y team's roadmap unblocked + I documented the pattern so it wouldn't recur" |

**Staff bar requires**: scope ≥ team, influence ≥ direct authority, result = quantified + second-order.

## Universal Red Flags (lower bar at every company)

- Saying "we" when describing YOUR specific contribution
- No quantified result — "it went well" is not a result
- Situaton is team-level, not org/company-level (insufficient scope for Staff)
- Story has no tension — if there was no hard part, pick a different story
- Blaming others for the failure in failure stories — no ownership, no learning
- Resolution took only 1 meeting — Staff problems are multi-week, multi-stakeholder
